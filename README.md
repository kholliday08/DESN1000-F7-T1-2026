# DESN1000-F7-T1-2026
This is the repository for the code for our Waste Sorting Robot project.
We will be using Python to code the Arduino. All modifications made to the code should be supported by comments where appropriate and well named commits to Github.
## Description
This project involves building a robot that utilises ultrasonic and IR sensors and Pixycam to sort recyclable plastic waste by types, specifically PET, PP and HDPE.
These pieces of waste will be represented by green, blue and red foam balls respectively.
## Features
- Object detection using an ultrasonic sensor.
- Identifying ball colour (red, green, blue, yellow).
- Utilises line-following to move around an environment.
- Real-time distance measurement.
- Transportation of objects from a central collection zone to a deposit zone.
- Reflects real-world waste-handling industries.
## Technologies used
- Duinotech UNO R3 (Arduino compatible)
- Python
- Ultrasonic sensor
- Pixycam
- Breadboard
